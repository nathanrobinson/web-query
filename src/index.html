Usage:

web-query [-d|--debug] [-q|--query "selector"] "url"

-q | --query: Query the document using a valid CSS selector and print the innerText. Multiple -q and corresponding queries can be provided and matches will be printed on new lines. web-query will automatically exit after the last query is run.

-d | --debug: Open the dev tools.

url: The url to open. "https://" is automatically prepended to the url if it is not included.
